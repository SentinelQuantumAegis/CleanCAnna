
# CleanCAnna — Static Site

This folder contains a single-page site ready to deploy as `index.html`.

## Quick publish options

### GitHub Pages
1) Create a new repo (e.g., `cleancanna-site`).
2) Upload the files in this folder (`index.html`, images).
3) In the repo: Settings → Pages → Deploy from branch → `main` → `/ (root)`.
4) Your site will be live at `https://<username>.github.io/cleancanna-site/`.

### Netlify Drop
1) Go to https://app.netlify.com/drop
2) Drag-and-drop this folder.
3) Use the generated `https://...netlify.app` URL in Stripe.
